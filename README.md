# Delta Force Game UI / 三角洲行动游戏界面

一个响应式的三角洲行动游戏信息展示界面 / A responsive Delta Force game information display interface.

## 功能特点 / Features

- 🎮 **游戏信息展示** / Game Information Display
- 🌍 **双语支持** / Bilingual Support (Chinese/English)
- 📱 **响应式设计** / Responsive Design
- ✨ **交互式问答** / Interactive Q&A
- 🎨 **现代化UI设计** / Modern UI Design

## 项目结构 / Project Structure

```
UI/
├── index.html          # 主页面文件 / Main HTML file
├── styles.css          # 样式文件 / CSS styles
├── script.js           # 交互脚本 / JavaScript functionality
└── README.md           # 项目说明 / Project documentation
```

## 如何使用 / How to Use

1. 克隆或下载项目 / Clone or download the project
2. 在浏览器中打开 `index.html` / Open `index.html` in a browser
3. 或使用本地服务器 / Or use a local server:
   ```bash
   python3 -m http.server 8080
   # 然后访问 / Then visit: http://localhost:8080
   ```

## 技术栈 / Tech Stack

- **HTML5** - 页面结构 / Page structure
- **CSS3** - 样式和动画 / Styling and animations
- **JavaScript (ES6+)** - 交互功能 / Interactive functionality
- **响应式设计** / Responsive design principles

## 特色功能 / Special Features

### 互动问答 / Interactive Q&A
点击"你还想了解什么？"按钮，获取更多关于三角洲行动游戏的信息。
Click the "What else would you like to know?" button to get more information about Delta Force games.

### 游戏系列信息 / Game Series Information
包含开发商、发布年份、游戏类型和支持平台等详细信息。
Includes detailed information about developer, release year, genre, and supported platforms.

### 现代化设计 / Modern Design
- 渐变背景和动画效果 / Gradient backgrounds and animations
- 悬停交互效果 / Hover interaction effects
- 平滑滚动和视差效果 / Smooth scrolling and parallax effects
- 移动端优化 / Mobile optimization
